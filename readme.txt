За стартиране на проекта трябва да се последват следните стъпки:
1. Първо трябва да се влезе в Client папката и да се напише "npm install".
2. След това "ng serve", за да се стартира Angular приложението.
3. После в папка Server трябва да се стартира sln файла с VisualStudio 2022 (тъй като е на .net 6). После се стартира проекта стандартно.

За FE частта се изисква node js инсталиран последна версия.
